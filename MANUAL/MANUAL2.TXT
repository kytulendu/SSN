                       Error Handling

     เนื่องจากโปรแกรมนี้มีขนาดใหญ่ และมีตัวแปรจำนวนมาก เกิด Error จึงมีได้ค่อยข่งมาก เพื่อความ
สะดวกในการติดตามแก้ไขปัญหา ได้จัดให้มี Routine ซึ่งคอยเก็บรายละเอียดของการเกิด Error ลงใน
File ชื่อ ERRLOG.DBF ซึ่งจะส่งกลับมาพร้อมกับรายงานที่ส่งส่วนกลาง เพื่อให้คณะผู้พัฒนาโปรแกรมสามารถ
แก้ไขได้ถูกต้อง อย่างไรก็ตม ยังมี Error อีกหลายกรณีที่เป็น Fatal error คือไม่สามารถแก้ไขหรือข้าม
ไปได้ routine ที่จัดไว้ก็ไม่สามารถจัดการได้ หากเกิดกรณีเช่นนี้ ผู้ใช้โปรดแจ้งให้ผู้พัฒนาโปรแกรมทราบ
รวมถึงรายละเอียดการทำงาน ขณะที่เกิด error ขึ้น

          การแก้ไขเบื้องต้นหากมี Error เกิดขึ้น คือ
     - สร้างไฟล์ดัชนีขึ้นใหม่ จากรายการบำรุงรักษาระบบ
     - สำรองข้อมูลไปเก็บที่ Diskette แล้วทำการ Install โปรแกรมใหม่ แล้งจึง Restore
       ข้อมูลเข้ามาทำงาน จากรายการบำรุงรักษาระบบ
     - Check Hard disk โดยใช้ CHKDSK  หรือ Utility อื่นๆ
     - ตรวจค้นหา Virus
